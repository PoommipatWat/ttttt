เครื่องรักษาระดับลูกปิงปอง

โปรเจคของกลุ่มพวกเรา คือ เครื่องรักษาระดับลูกปิงปอง โดยอาศัยกลักการ PID ซึ่งเป็นระบบที่ใช้ในการควบคุมความเสถียรของอุปกรณ์ทางอิเล็กทรอนิคต่างๆ เช่น มอเตอร์ หรือ heater เพื่อให้ค่าอยู่ในระดับที่เราต้องการอยู่เสมอ โดยเครื่องรักษาระดับลูกปิงปอง จะมีการตรวจจับระดับความสูงของลูกปิงปองด้วยเซนเซอร์ ultrasonic นอกจากนี้โปรเจคของพวกเรายังสามารถสั่งการได้จาก website โดยมีการส่งข้อมูลด้วย MQTT

Part การออกแบบด้วย Solidworks

ใน Part ชิ้นงานบางส่วนเราได้มีการนำไฟล์สำเร็จจาก thingiverse มาดัดแปลงเพื่อให้เข้ากับการใช้งานมากขึ้น และเราได้นำโมเดลเหล่านี้ไปปริ้นด้วยเครื่อง 3d printer

การออกแบบ PCB Board

เราเริ่มจากการออกแบบ schematic diagram และนำไปออกแบบ PCB board โดยใช้โปรแกรม easy eda จากนั้นเราได้สั่งทำบอร์ดจากเว็บไซต์ pcbway

การออกแบบ Code

เราได้เริ่มจากการทดสอบ ultrasonic sensor เป้นอย่างแรก พบว่ามีความคลาดเคลื่อนอยู่ที่ 1 เซนติเมตร หลังจากนั้นเราได้ศึกษาการปรับค่า PID ให้ลูกปิงปองไปยังตำแหน่งที่เราต้องการให้ได้ใกล้เคียงมากที่สุด เมื่อเราปรับค่าคนลูกปิงปองมีความนิ่งและอยู่บนตำแหน่งที่ต้องการได้แล้ว เราพบปัญหาคือ PID เกิดการ overshoot มากเกินไป เราจึงลองปรับค่า KD เพิ่มเติม ผลลัพธ์ คือ สามารถลดอาการ overshoot ได้จริง แต่ค่า error จากจุดที่เราต้องการมีค่ามากขึ้น จึงไม่คุ้มกับการเพิ่มค่า KD เราจึงยอมให้เกิด overshoot และรักษาค่า error ที่เหมาะสม
